# World-Cup-2023-Analysis
🏏 ICC Cricket World Cup 2023 Data Analysis (Python | Google Colab)
This project is a Data Analysis of the ICC Cricket World Cup 2023, developed using Python in Google Colab. It explores match data, player performance, and team statistics to uncover meaningful insights and trends from the tournament.
🔗 Open Notebook:
https://colab.research.google.com/drive/1amwqVuba024eH7owwlErY7JE0vaQ0Rlo

🚀 Project Overview
The goal of this project is to perform Exploratory Data Analysis (EDA) on World Cup 2023 data and derive insights such as:

Team performance comparison
Player statistics (batting & bowling)
Match outcomes and trends
Tournament-level insights


✨ Features

📊 Data cleaning and preprocessing
🏏 Team-wise performance analysis
🎯 Player statistics (runs, wickets, strike rate, etc.)
📈 Match trends and result analysis
📉 Visualization of key metrics
📌 Insight generation from real match data


🛠️ Technologies Used

Python
Google Colab (Jupyter Notebook)
Pandas (data analysis)
NumPy (numerical operations)
Matplotlib / Seaborn (data visualization)
